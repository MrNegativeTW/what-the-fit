## 功能

- 依照選擇的部位隨機產生穿搭
- 可整套重抽，或只重抽單一部位
- 可管理衣物、部位、品牌、顏色、場合
- 支援衣櫃搜尋
- 離線可用（資料只存在裝置）

## 本機執行

1. 下載專案
2. 用 Android Studio 開啟
3. 建置 Debug APK

```bash
./gradlew :app:assembleDebug
```

4. 安裝到模擬器或裝置

```bash
./gradlew :app:installDebug
```
